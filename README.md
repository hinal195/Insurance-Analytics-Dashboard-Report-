# 📊 Advanced Insurance Policy Analysis Dashboard

An end-to-end **Power BI** project designed to transform raw insurance data into actionable business insights through interactive dashboards and data visualization.

## 📌 Project Overview

The Advanced Insurance Policy Analysis Dashboard helps insurance companies analyze policy performance, premium trends, investment growth, customer demographics, and sales hierarchy. The dashboard enables stakeholders to monitor business performance and make informed, data-driven decisions using interactive reports.

---

## 🎯 Objectives

- Analyze insurance policy performance.
- Monitor premium collection and payment status.
- Compare investment value with maturity value.
- Evaluate annual premium against protection value.
- Analyze sales performance across agents and managers.
- Identify trends using interactive visualizations.
